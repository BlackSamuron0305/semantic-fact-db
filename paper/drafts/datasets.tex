\section{Datasets}
\label{sec:datasets}

We use both synthetic and real-world datasets for evaluation.

\subsection{Synthetic Datasets}
\label{sec:datasets:synthetic}

Synthetic datasets are generated with controlled properties:
\begin{itemize}
    \item Varying fact arity ($k = 2, 3, 4, 5$).
    \item Varying context depth ($d = 1, 2, 3, 4, 5$).
    \item Varying graph density (sparse to fully connected).
    \item Controlled entity and relation counts.
\end{itemize}

\subsection{Real-World Datasets}
\label{sec:datasets:real}

Real-world datasets include subsets of widely-used knowledge graphs:
\begin{itemize}
    \item \textbf{LUBM}: Lehigh University Benchmark~\cite{lubm2005}
          with university domain ontology.
    \item \textbf{DBpedia}: Extracted structured data from Wikipedia.
    \item \textbf{Wikidata}: Collaborative knowledge base.
\end{itemize}
