The experimental results demonstrate that:
(1) SheafDatabase achieves competitive latency with
KnowledgeGraph while offering richer semantic filtering;
(2) both engines scale linearly with dataset size;
(3) the sheaf representation reduces memory overhead;
(4) the common optimizer provides equivalent plan quality
across both backends.
