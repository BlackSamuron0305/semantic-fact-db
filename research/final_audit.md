## Issues Found

1. **Intersection closure not called on incremental topology path** — topologies built via incremental insert are not valid topologies until `intersection_closure()` is called (done in `verify()`, but not on every insert). This is acceptable: the missing intersections are not required for query operations.

2. **Global section cache invalidated on every insert** — still O(1) invalidation (dict clear), which is fine for correctness. Incremental update not implemented (future work).

3. **ConsistencyChecker restriction composition fails on open sets without sections** (𝕌 and ∅) — pre-existing issue; does not affect query correctness.

4. **No explicit cocycle condition (d² = 0)** — the gluing and locality checks cover this implicitly but not via formal cohomological computation.
