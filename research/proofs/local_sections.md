# Local Sections

## Definition

Let $F: C^{\text{op}} \to \mathbf{Set}$ be a presheaf on a poset
$(C, \leq)$.  A **local section** (or simply a **section**) over an
open set $U \subseteq C$ (in the Alexandrov topology) is an element
$s \in F(U)$.

In the poset setting, an open set $U$ corresponds to a **down-set**:
$u \in U$ and $v \leq u$ implies $v \in U$.  The maximal open sets are
the **principal down-sets** $\downarrow c = \{ d \in C \mid d \leq c \}$
generated by a single context $c$.

A **local section** over $c$ is therefore an element of $F(c)$,
interpreting $F(c)$ as $F(\downarrow c)$.

## Interpretation for SFDB

A local section in SFDB is a ``SemanticFact`` whose context field is $c$.
The fact is "local" in the sense that it is only guaranteed to hold within
the scope $c$ and its sub-contexts (i.e. within $\downarrow c$).

If we ask "does fact $f$ hold in context $c$?" we check whether there
exists a section $s \in F(c)$ (by retrieval or restriction from a
broader context) such that $s$ matches $f$.

## Properties

1. **Refinement**: If $s \in F(c_2)$ and $c_1 \leq c_2$, then
   $\rho_{c_2, c_1}(s) \in F(c_1)$.  A section valid in a broader context
   is also valid (in restricted form) in any narrower context.

2. **Maximality**: A section $s \in F(c)$ is **maximal** if there is
   no $d > c$ and $t \in F(d)$ such that $\rho_{d, c}(t) = s$.
   Maximal sections represent facts stated at the most general level
   possible.
