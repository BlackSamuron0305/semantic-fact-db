# Finite Topological Spaces and the Alexandrov Topology

## Definition (Finite Topological Space)

A **finite topological space** is a topological space $(X, \mathcal{T})$ where
$X$ is a finite set and $\mathcal{T} \subseteq \mathcal{P}(X)$ satisfies the
usual axioms:

1. $\emptyset, X \in \mathcal{T}$.
2. $\mathcal{T}$ is closed under finite intersections.
3. $\mathcal{T}$ is closed under arbitrary unions (which, because $X$ is
   finite, is the same as closure under finite unions).

## Definition (Alexandrov Topology)

An **Alexandrov topology** is a topology in which arbitrary intersections
of open sets are open.  Every finite topological space is trivially
Alexandrov because all intersections are finite.  More importantly, there
is a bijection between Alexandrov topologies on $X$ and preorders
(reflexive, transitive relations) on $X$:

- Given a preorder $\leq$ on $X$, define the **Alexandrov topology**
  by taking the **downward-closed** sets as opens:
  \[
  U \subseteq X \text{ is open } \iff \forall x \in U,\; \forall y \leq x,\; y \in U.
  \]
  Equivalently, every point $x$ has a smallest neighbourhood
  $U_x = \{ y \mid y \leq x \}$.

- Conversely, given an Alexandrov topology $\mathcal{T}$, define
  $x \leq y$ iff $x \in \overline{\{y\}}$ (i.e. $x$ is in the closure
  of $\{y\}$), which for Alexandrov spaces is a preorder.

## Why This Matters for SFDB

The **context poset** $(C, \leq)$ in SFDB is a finite partially ordered set.
Every poset induces an Alexandrov topology where the open sets are the
**downward-closed** subsets (down-sets).  A set $U \subseteq C$ is open
iff whenever $c \in U$ and $d \leq c$, then $d \in U$.

This topological reading lets us:

- Define a **presheaf** $F: \mathcal{T}^{\text{op}} \to \mathbf{Set}$
  on the Alexandrov topology generated by the context poset.
- Use the **sheaf condition** (locality + gluing) for contextual
  reasoning.
- Identify **global sections** with facts that hold in every context.
